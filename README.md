# Safari Bingo Tanzania

Safari Bingo Tanzania is a static offline-ready web app for tracking animal sightings on safari.

## Features

- Mark sightings for 19 popular safari animals found in Tanzania
- View local photos and short descriptions for each animal
- Play with a random 4x4 bingo card that always includes the Big Five
- Keep progress in local storage
- Continue using the app offline after the first successful load

## Local Preview

From the project directory:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deploy

This repo is ready to connect to Vercel as a static site.
